# Vett CLI

CLI for the [Vett](https://vett.sh) secure agent skill registry.

## Installation

```bash
npm install -g vett
```

Or use directly with npx:

```bash
npx vett --help
```

## Usage

```bash
# Show help
vett --help

# Show version
vett --version

# Install a skill
vett install <skill>

# Search for skills
vett search <query>
```

## Commands

| Command | Description |
|---------|-------------|
| `install <skill>` | Install a skill from the registry |
| `search <query>` | Search for skills |
| `info <skill>` | Show information about a skill |
